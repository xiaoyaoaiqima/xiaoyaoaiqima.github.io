format简介

  • 解决浮点数精度丢失

    double pi = 3.14159265358979;
    cout << format("{}\n", pi);

  • 取代sprintf

    char buff[50];
    auto end = format_to_n(buff, sizeof(buff), "{}", pi);
    end是一个迭代器，指向缓冲区最后一个字符
    用于把pi填入buff
    通过 string 访问缓冲区中的数据
    string str(buff, end - buff);

  • 与STL容器完美结合

    vector <char> v;
    std::back_inserter可以将元素插入到末尾
    format_to(back_inserter(v), "{}", "1999");
    返回{"1","9","9","9"}

  • 其他

    计算format之后所需空间
    auto size = formatted_size("{}", 42);
    给vector创造合适的空间
    vector <char> buf(size);
    data直接指向数据，避免其他开销
    format_to(buf.data(), "{}", 42);
